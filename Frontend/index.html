Results

Six real experiments, reported honestly

Rather than one model tuned to look good, we tested six different hormone-outcome relationships against real NHANES 2017–March 2020 data — and kept the negative results, not just the flattering ones.

ModelPopulationBest resultFinding
Menopause status All women 97.8% accuracy Barely beats guessing by age alone — the real value is the honest edge-case limitation, not the headline number.
Hormone therapy detection All women 77.6% recall Strongest result — a direct, mechanistic hormone relationship.
Infertility history Reproductive-age 61.2% recall Real partial signal; adding more features did not improve it.
Menstrual irregularity Under 40 60.0% recall Metabolic markers (insulin, glucose) meaningfully improved a weak baseline.
Age-at-menopause timing Post-menopausal no signal A single hormone snapshot can't predict future timing — tested with linear and non-linear models.
Premature menopause Post-menopausal not viable Only 5 true cases in this NHANES cycle — a data-volume limit, not a modeling failure.
